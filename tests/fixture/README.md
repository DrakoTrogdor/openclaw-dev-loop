# greeter

A simple CLI that greets users by name.

## Usage

```
python -m greeter --name <name> [--shout] [--times <n>] [--reverse]
```

## Options

- `--name`     Name to greet (required)
- `--shout`    Print greeting in uppercase
- `--times`    Repeat the greeting N times (default: 1)
- `--reverse`  Print the greeting reversed

## Build & Test

```bash
./build.sh
```

## Requirements

- Python 3.8+
